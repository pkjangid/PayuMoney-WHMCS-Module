# PayuMoney-WHMCS-Module
PayuMoney WHMCS for integration of Payment gateway in WHMCS

1. Upload/place the gateway file payumoney.php which is in gateway folder into <whmcs root directory>/module/gateway/

2. Upload/Place the call back file payumoney.php which is in gateway folder into <whmcs root directory>/module/gateway/callback/

3. Log in as WHMCS Administrator & navigate to Payment Gateway option.
   <menu bar> -> Setup -> Payments -> Payment gateways

4. Select PayUMoney from dropdown in Payment Gateway setup page.

5. Input the valid details provided from PayUMoney.

6. Input the Success URL &  Failure URL are same 

<whmcs client-area url/modules/gateways/callback/payumoney.php
